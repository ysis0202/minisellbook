## [v1.0.0] - 2025-10-14 (Initial Release)

### ✨ 추가
- **기본 재정 관리**: 수입/지출 관리
- **월간 캘린더**: 날짜별 거래 내역
- **통계 & 분석**: 월별/카테고리별 분석
- **검색 기능**: 키워드 및 날짜 범위 검색
- **Google OAuth**: Google 소셜 로그인

### 🛠 기술 스택
- Next.js 15.5
- React 19.2
- TypeScript 5.6
- Supabase (PostgreSQL + Auth)
- Tailwind CSS
- SWR
